### 8.3.12 sigmoid

```c
double sigmoid(double x)
```

**功能介绍**

计算双精度浮点的 sigmoid 函数的取值。

- 参数 x 的取值范围在（-10，10）时：最大 ULP 为 124。
- 参数 x 取其他范围时：最大绝对误差 5e−05。

**参数解释**

| 参数名称 | 说明 |
|----------|------|
| x | 输入参数。 |

**返回值**

| 返回值类型 | 说明 |
|------------|------|
| double | sigmoid 函数的取值，其中：<br>- 当 x 小于 -10 时，返回 0。<br>- 当 x 大于 10 时，返回 1。<br>- 当 x 取 0 时，返回 0.5。<br>- 当 x 取 +∞ 时，返回 1。<br>- 当 x 取 −∞ 时，返回 0。<br>- 当 x 取 NaN 时，返回 NaN。 |

**注意事项**

该接口支持原位操作。

**使用示例 1**

```c
__device__ void test(double *a, double *b, int len)
{
    for (int i = 0; i < len; i++) {
        b[i]= sigmoid(a[i]) + 1;
    }
}
```

**使用示例 2**

```c
__device__ void test(double *a, int len)
{
    for (int i = 0; i < len; i++) {
        a[i]= sigmoid(a[i]) + 1;      // 原位操作
    }
}
```

### 8.4.1 simd_exp

```c
floatv16 simd_exp(floatv16 x)
```

**功能介绍**

计算 e 为底数的指数向量，一次计算 16 个 float。最大 ULP 为 5。

**参数解释**

| 参数名称 | 说明 |
|----------|------|
| x | 输入向量，float 个数为 16。 |

**返回值**

| 返回值类型 | 说明 |
|------------|------|
| floatv16 | e 为底数的指数向量，其中：<br>- 输入向量的某个值小于等于 -87.33，该位置对应返回 0。<br>- 输入向量的某个值大于等于 88.72，该位置对应返回 +∞。<br>- 输入向量的某个值是 +∞，该位置对应返回 +∞。<br>- 输入向量的某个值是 −∞，该位置对应返回 0。<br>- 输入向量的某个值是 NaN，该位置对应返回 NaN。 |

**注意事项**

该接口支持原位操作。

**使用示例 1**

```c
__device__ floatv16 simd_exp_test()
{
    float input_arr[16] = {0};
    floatv16 input;
    floatv16 result;

    for (int i = 0; i < 16; i++) {
        input_arr[i] = -87.33 + (176.0 / 16) * i;
    }
    simd_load(input, input_arr);

    result = simd_exp(input);
    return result;
}
```

**使用示例 2**

```c
__device__ floatv16 simd_exp_test()

    float input_arr[16] = {0};

    return input;
}
```

|----------|------|
| x | 输入数组：首地址 64 字节对齐，float 个数为 128。<br>计算结果：以 e 为底数的指数数组，写回输入参数对应位置，其中：<br>- 输入数组的某个值小于等于 -87.33，该位置对应返回 0。<br>- 输入数组的某个值大于等于 88.72，该位置对应返回 +∞。<br>- 输入数组的某个值是 +∞，该位置对应返回 +∞。<br>- 输入数组的某个值是 −∞，该位置对应返回 0。<br>- 输入数组的某个值是 NaN，该位置对应返回 NaN。 |

**返回值**

无。

**注意事项**

该接口支持原位操作。

**使用示例**

```c
__device__ void simd_exp128_test()

    float input_arr[128] __attribute__((aligned(64))) = {0};   // 64B aligned
```
