## 16.4.2.4 双缓冲

### 16.4.2.4.1 实现原理

本章主要介绍如何在广播的基础上，使用双缓冲优化策略，提升算子性能。

#### 优化原理

主机与设备间的数据传输，可以与设备上 Kernel 的执行并行，互不影响。这说明：主机端可以通过异步的数据传输接口发起数据传输后，立即发起 Kernel 的执行操作，实现访存和计算的并行，进一步缩短程序耗时。使用 Global 存储与 SPM 存储间的非阻塞型 DMA 接口，可以实现计算和数据传输的并行。

为实现计算和数据传输的并行，两者的数据不能相互依赖，即计算过程的数据不能依赖于同一时刻正在传输的数据。因此，需要在 SPM 存储空间上，设置至少 2 个数据缓冲区：

- **计算缓冲区**：存储计算过程依赖的数据。
- **传输缓冲区**：存储异步传输过程依赖的数据。

当计算和数据传输都完成后，再交换两个缓冲区的数据，即可始终确保计算过程与数据传输过程的数据无依赖。

> 图1 双缓冲的实现原理

#### 代码逻辑

双缓冲通过 `tecoKernelGemmDoubleBufferImpl` 函数实现，该函数在广播的基础上，使用双缓冲进行性能优化。

1. **创建缓冲区，并初始化指针**

   a. 为矩阵 `A`、`B` 和结果矩阵 `C` 的子块分配 SPM 缓冲区，用于后续的计算、数据暂存等操作。

   b. 初始化两组指针：

   - `pDA` 和 `pCA` 分别指向步骤 a 分配的矩阵 `A` 的传输缓冲区 `LocalA` 和计算缓冲区 `LocalCompA`。
   - `pDB` 和 `pCB` 分别指向步骤 a 分配的矩阵 `B` 的传输缓冲区 `LocalB` 和计算缓冲区 `LocalCompB`。

2. **创建广播句柄，并初始化广播相关的数据结构和变量**

   调用 `BroadcastHandle` 创建广播句柄，并初始化广播相关的数据结构和变量，包括：行广播线程组、列广播线程组、跨步对象、以及传输缓冲区和计算缓冲区。

3. **初始化矩阵乘接口**

   调用 `matmul_init` 初始化矩阵乘接口，并配置输入和输出数据的类型。

4. **进行非阻塞型广播**

   a. 计算用于广播的线程索引 `typeAThreads` 和 `typeBThreads`，并创建线程组 `col_thread_group` 和 `row_thread_group` 以及相应的广播句柄 `col_handle` 和 `row_handle`。

   b. 调用 `sync_threads()` 函数，同步所有相关线程。

   c. 通过 `broadcast_async` 函数，分别对矩阵 `A` 和 `B` 进行非阻塞型广播操作。

   - 将矩阵 A 的数据，按照指定的块大小 `BsizeA` 和线程数量 `typeAThreads`，从 Global 存储空间广播到对应的缓冲区（`pDA` 指向的缓冲区）。
   - 将矩阵 B 的数据，按照指定的块大小 `BsizeB` 和线程数量 `typeBThreads`，从 Global 存储空间广播到对应的缓冲区（`pDB` 指向的缓冲区）。

5. **执行矩阵乘法计算，并等待相关操作的执行完成**

   - 进行计算并更新指针：循环遍历所有需要处理的矩阵 `A` 和 `B` 的组合。

     a. 在每次外层循环中，先通过 `idx` 计算出当前在 `M`、`N` 维度上的块索引 `index_M` 和 `index_N`。再通过 `index_K` 遍历矩阵在 `K` 维度上的块。

     b. 在循环内部，根据当前的块索引 `index_M`、`index_K` 和 `index_N`，更新指向矩阵 `A` 和 `B` 当前块的指针 `pCurrA` 和 `pCurrB`，以及指向下一个块（在 `K` 维度上的下一个位置，或者根据边界情况进行特殊处理后的下一个块位置）的指针 `pNextA` 和 `pNextB`。

   - 交换计算缓冲区和传输缓冲区指针：

     - 对于矩阵 `A`：交换 `pCA` 和 `pDA`。
     - 对于矩阵 `B`：交换 `pCB` 和 `pDB`。

6. **等待上一次缓存任务完成**

   调用 `broadcast_wait` 和 `matmul_wait_loading_input` 等函数，等待矩阵 A 的广播操作和矩阵乘法的输入数据加载完成，确保数据准备好后再进行下一步操作。

7. **继续发起下一次访存任务，同时并行矩阵乘计算**

   在等待当前块的相关操作完成后，使用 `broadcast_async` 函数异步广播矩阵 A 和 B 的下一个块数据到对应的缓冲区，提前准备好后续计算所需的数据。

8. **执行矩阵乘法计算，并存储计算结果**

示例代码如下：

```cpp
__global__ void tecoKernelGemmDoubleBufferImpl(GEMMArgs pGemm)
{
    // 划分线程的行和列
    const int rid = threadIdx / 8;
    const int cid = threadIdx % 8;


    // 从 GEMMArgs 结构体中获取矩阵的维度信息
    const int M = pGemm.m;
    const int N = pGemm.n;
    const int K = pGemm.k;


    //   从 GEMMArgs 结构体中获取矩阵的存储步长
    const int lda = pGemm.lda;
    const int ldb = pGemm.ldb;
    const int ldc = pGemm.ldc;


    // 指向输入矩阵 A、B 和输出矩阵 C 的指针
    const half *A = (const half *)pGemm.A;
    const half *B = (const half *)pGemm.B;
    half *C = (half *)pGemm.C;


    // 外部块的大小
    const int bM = pGemm.bM;
    const int bN = pGemm.bN;
    const int bK = pGemm.bK;


    // 外部块的数量
    int nM = M / bM;
    int nN = N / bN;
    int nK = K / bK;


    // 内部块的大小
    int LocalbM = bM / 8;
    int LocalbN = bN / 4;
    int LocalbK = bK;


    // 内部块的位置
    const half *pA = A + cid * LocalbM * lda;
    const half *pB = B + rid * LocalbN;
    half *pC = C + cid * LocalbM * ldc + rid * LocalbN;


    // A、B、C 的当前块和下一个块的指针
    const half *pCurrA, *pNextA;
    const half *pCurrB, *pNextB;
    half *pCurrC, *pNextC;


    // 计算内存步长
    const int LenA = LocalbM * LocalbK * sizeof(half);
    const int BsizeA = LocalbK * sizeof(half);
    const int StrideA = (lda - LocalbK) * sizeof(half);


    const int LenB = LocalbK * LocalbN * sizeof(half);
    const int BsizeB = LocalbN * sizeof(half);
    const int StrideB = (ldb - LocalbN) * sizeof(half);


    const int LenC = LocalbM * LocalbN * sizeof(half);
    const int BsizeC = LocalbN * sizeof(half);
    const int StrideC = (ldc - LocalbN) * sizeof(half);


    int spm_size = 2 * LenA + 3 * LenB + LenC + LenC / sizeof(half) * sizeof(float) * 2;


    // 为 A、B 的子块和结果 C 分配SPM缓冲区，包括双缓冲区
    half *LocalA = (half *)malloc(LenA * 2);
    half *LocalCompA = LocalA + LocalbM * LocalbK;
    half *LocalB = (half *)malloc(LenB * 3);
    half *LocalCompB = LocalB + LocalbK * LocalbN;
    half *LocalTB = LocalB + LocalbK * LocalbN * 2;
    half *LocalC = (half *)malloc(LenC);
    float *LocalCompC = (float *)malloc(LocalbM * LocalbN * sizeof(float) * 2);
    float *tempC = LocalCompC + LocalbM * LocalbN;


    // 中间计算的临时变量
    half *pTemp;


    // 矩阵A、B 访存和计算缓冲区的指针
    half *pDA = LocalA, *pCA = LocalCompA;
    half *pDB = LocalB, *pCB = LocalCompB;


    int typeAThreads = (3 - cid % 4) * 8 + cid;   // 列广播
    int typeBThreads = rid * 8 + rid;             // 行广播


    // 遍历块和块内元素的循环变量
    int im, in, ik;
    int index_M, index_N, index_K;


    // B 置换的准备
    floatv16 vtb[32];


    // 开始性能采样
    perf_start();


    // 初始化矩阵乘法操作
    MatmulHandle handleMMA;
    matmul_init(handleMMA, MatmulHalfToFloat);


    // 广播的准备
    uint64_t mask = 0;
    mask = 0x01010101 << cid;
    ThreadGroup col_thread_group(mask);


    mask = 0xFF << (8 * rid);
    ThreadGroup row_thread_group(mask);


    // 创建广播句柄
    BroadcastHandle col_handle(&col_thread_group);
    BroadcastHandle row_handle(&row_thread_group);
    sync_threads();


    // 非阻塞广播矩阵 A 和 B
    broadcast_async(pDA, (void *)pA, LenA, BsizeA, {0, (size_t)StrideA}, typeAThreads, BroadcastGlobalToSpm, col_handle);
    broadcast_async(pDB, (void *)pB, LenB, BsizeB, {0, (size_t)StrideB}, typeBThreads, BroadcastGlobalToSpm, row_handle);


    // 遍历矩阵 A 和 B 的块，计算块积并累加到矩阵 C 中
    for (int idx = 0; idx < nM * nN; ++idx) {
        // 计算 M、N 维度的块索引
        index_M = idx / nN;
        index_N = idx % nN;
        // 遍历 K 维度，将 A 和 B 的块相乘并将结果累加到 TempC 中
        for (index_K = 0; index_K < nK; ++index_K) {
            // 更新矩阵 A 和 B 的当前块的指针
            pCurrA = pA + index_M * bM * lda + index_K * bK;   // A[index_M][index_K]
            pCurrB = pB + index_K * bK * ldb + index_N * bN;   // B[index_K][index_N]
            // 定位 pNextA 和 pNextB
            if (index_K < nK - 1) {
                pNextA = pA + index_M * bM * lda + (index_K + 1) * bK;     // A[index_M][index_K+1]
                pNextB = pB + (index_K + 1) * bK * ldb + index_N * bN;     // B[index_K+1][index_N]
            } else {
                if (index_N == nN - 1) {
                    pNextA = pA + (index_M + 1) * bM * lda;    // A[index_M+1][0]
                    pNextB = pB;                         // B[0][0]
                    if (idx == (nM * nN - 1)) pNextA = pA;
                } else {
                    pNextA = pA + index_M * bM * lda;   // A[index_M][0]
                    pNextB = pB + (index_N + 1) * bN;   // B[0][index_N+1]
                }
            }


            // 交换 pCA 和 pDA
            pTemp = pCA;
            pCA = pDA;
            pDA = pTemp;


            // 交换 pCB 和 pDB
            pTemp = pCB;
            pCB = pDB;
            pDB = pTemp;


            // 等待 A 广播和矩阵乘法数据加载
            broadcast_wait(col_handle);
            matmul_wait_loading_input(handleMMA);


            // 同步列线程组内的线程
            sync_threads(col_thread_group);


            // 非阻塞广播矩阵 A 的下一个块
            broadcast_async(pDA, (void *)pNextA, LenA, BsizeA, {0, (size_t)StrideA}, typeAThreads, BroadcastGlobalToSpm,
                              col_handle);


            // 对矩阵 B 进行相同操作
            broadcast_wait(row_handle);
            matmul_wait_loading_weight(handleMMA);


            sync_threads(row_thread_group);


            broadcast_async(pDB, (void *)pNextB, LenB, BsizeB, {0, (size_t)StrideB}, typeBThreads, BroadcastGlobalToSpm,
                              row_handle);


            // 置换矩阵 B
            for (ik = 0; ik < LocalbK; ik += 8) {
                simd_load(vtb[0], (float *)(pCB + ik * LocalbN));
                simd_load(vtb[1], (float *)(pCB + ik * LocalbN + 32));
                simd_load(vtb[2], (float *)(pCB + (ik + 1) * LocalbN));
                simd_load(vtb[3], (float *)(pCB + (ik + 1) * LocalbN + 32));
                simd_load(vtb[4], (float *)(pCB + (ik + 2) * LocalbN));
                simd_load(vtb[5], (float *)(pCB + (ik + 2) * LocalbN + 32));
                simd_load(vtb[6], (float *)(pCB + (ik + 3) * LocalbN));
                simd_load(vtb[7], (float *)(pCB + (ik + 3) * LocalbN + 32));
                simd_load(vtb[8], (float *)(pCB + (ik + 4) * LocalbN));
                simd_load(vtb[9], (float *)(pCB + (ik + 4) * LocalbN + 32));
                simd_load(vtb[10], (float *)(pCB + (ik + 5) * LocalbN));
                simd_load(vtb[11], (float *)(pCB + (ik + 5) * LocalbN + 32));
                simd_load(vtb[12], (float *)(pCB + (ik + 6) * LocalbN));
                simd_load(vtb[13], (float *)(pCB + (ik + 6) * LocalbN + 32));
                simd_load(vtb[14], (float *)(pCB + (ik + 7) * LocalbN));
                simd_load(vtb[15], (float *)(pCB + (ik + 7) * LocalbN + 32));
                simd_store(vtb[0], (float *)(LocalTB + ik * 32));
                simd_store(vtb[1], (float *)(LocalTB + 32 * LocalbK + ik * 32));
                simd_store(vtb[2], (float *)(LocalTB + (ik + 1) * 32));
                simd_store(vtb[3], (float *)(LocalTB + 32 * LocalbK + (ik + 1) * 32));
                simd_store(vtb[4], (float *)(LocalTB + (ik + 2) * 32));
                simd_store(vtb[5], (float *)(LocalTB + 32 * LocalbK + (ik + 2) * 32));
                simd_store(vtb[6], (float *)(LocalTB + (ik + 3) * 32));
                simd_store(vtb[7], (float *)(LocalTB + 32 * LocalbK + (ik + 3) * 32));
                simd_store(vtb[8], (float *)(LocalTB + (ik + 4) * 32));
                simd_store(vtb[9], (float *)(LocalTB + 32 * LocalbK + (ik + 4) * 32));
                simd_store(vtb[10], (float *)(LocalTB + (ik + 5) * 32));
                simd_store(vtb[11], (float *)(LocalTB + 32 * LocalbK + (ik + 5) * 32));
                simd_store(vtb[12], (float *)(LocalTB + (ik + 6) * 32));
                simd_store(vtb[13], (float *)(LocalTB + 32 * LocalbK + (ik + 6) * 32));
                simd_store(vtb[14], (float *)(LocalTB + (ik + 7) * 32));
                simd_store(vtb[15], (float *)(LocalTB + 32 * LocalbK + (ik + 7) * 32));
            }


            if (index_K < nK - 1) {
                for (ik = 0; ik + 32 - 1 < LocalbK; ik += 32) {
                    // 执行实际的矩阵乘法
                    matmul_set_flushing_output(handleMMA, false);
                    matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, 0);
                    matmul_load_weight(handleMMA, LocalTB + ik * LocalbN / 2, MatmulK32, MatmulN32);
                    matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);


                    matmul_set_flushing_output(handleMMA, false);
                    matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, LocalbM);
                    matmul_load_weight(handleMMA,
                                        LocalTB + LocalbN * LocalbK / 2 + ik * LocalbN / 2,
                                        MatmulK32, MatmulN32);
                    matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);
                }
            } else {
                for (ik = 0; ik + 64 - 1 < LocalbK; ik += 32) {
                    matmul_set_flushing_output(handleMMA, false);
                    matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, 0);
                    matmul_load_weight(handleMMA, LocalTB + ik * LocalbN / 2, MatmulK32, MatmulN32);
                    matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);


                    matmul_set_flushing_output(handleMMA, false);
                    matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, LocalbM);
                    matmul_load_weight(handleMMA,
                                        LocalTB + LocalbN * LocalbK / 2 + ik * LocalbN / 2,
                                        MatmulK32, MatmulN32);
                    matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);
                }
                // 最后的矩阵乘法计算并存储结果
                matmul_set_flushing_output(handleMMA, false);
                matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, 0);
                matmul_load_weight(handleMMA, LocalTB + ik * LocalbN / 2, MatmulK32, MatmulN32);
                matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);


                matmul_set_flushing_output(handleMMA, true);
                matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, LocalbM);
                matmul_load_weight(handleMMA, LocalTB + LocalbN * LocalbK / 2 + ik * LocalbN / 2,
                                   MatmulK32, MatmulN32);
                matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);
                matmul_store(handleMMA, LocalCompC, LocalbM * 2, MatmulN32);
            }
        }   // loop index_K


        // 等待矩阵乘法操作完成
        matmul_wait(handleMMA);


        // 将矩阵 C 的已计算部分结果置换
        for (im = 0; im < LocalbM; im++) {
            simd_load(vtb[0], LocalCompC + im * 32);
            simd_load(vtb[1], LocalCompC + im * 32 + 16);
            simd_load(vtb[2], LocalCompC + (im + LocalbM) * 32);
            simd_load(vtb[3], LocalCompC + (im + LocalbM) * 32 + 16);
            simd_store(vtb[0], tempC + im * 64);
            simd_store(vtb[1], tempC + im * 64 + 16);
            simd_store(vtb[2], tempC + im * 64 + 32);
            simd_store(vtb[3], tempC + im * 64 + 48);
        }


        // 在存储结果之前，将计算结果进行类型转换
        batch_S2H(tempC, (half *)LocalC, 4096);


        // 将计算得到的块复制回全局内存
        memcpy_stride(pC + index_M * bM * ldc + index_N * bN, LocalC, LenC, BsizeC, {(size_t)StrideC, 0});
    }   // Loop idx


    // 性能采样
    perf_stop();
    // 打印出性能采样结果
    perf_print();


    // 释放内存
    free(LocalA);
    free(LocalB);
    free(LocalC);
    free(LocalCompC);
}
```

使用双缓冲优化策略的完整程序（`gemm_doublebuffer.scpp`）：

```cpp
#include <type_traits>
#include <sdaa_perf.h>


using namespace sdaa;


#define REDUITEM 32


// GEMMArgs包含矩阵乘法所需的参数
typedef struct GEMMArgs {

    int bN;    // 分块矩阵的列数

// 设备函数，将浮点数数组转换为半精度浮点数数组，并根据对齐情况进行处理
__device__ void align_batch_function(float* x, half* y, int len)
{
    // 定义向量变量
    floatv16 vx0, vx1, vx2, vx3, vx4, vx5, vx6, vx7;
    halfv16 vy0, vy1, vy2, vy3, vy4, vy5, vy6, vy7;
    int i = 0;


    // 外层循环，以 128 个元素为一组处理数据
    // 每次循环处理完一组后，指针移动 128 个元素位置继续处理下一组
    for (i = 0; i <= len - 128; i += 128) {
        // 使用 simd_load 函数加载数据
        simd_load(vx0, x);
        simd_load(vx1, x + 16);
        simd_load(vx2, x + 32);
        simd_load(vx3, x + 48);
        simd_load(vx4, x + 64);
        simd_load(vx5, x + 80);
        simd_load(vx6, x + 96);
        simd_load(vx7, x + 112);


        // 使用 simd_cvt_f2h 函数将加载的浮点型向量数据转换为半精度浮点型
        vy0 = simd_cvt_f2h(vx0);
        vy1 = simd_cvt_f2h(vx1);
        vy2 = simd_cvt_f2h(vx2);
        vy3 = simd_cvt_f2h(vx3);
        vy4 = simd_cvt_f2h(vx4);
        vy5 = simd_cvt_f2h(vx5);
        vy6 = simd_cvt_f2h(vx6);
        vy7 = simd_cvt_f2h(vx7);


        // 使用 simd_store 函数将转换后的半精度浮点型向量数据存储到内存中
        simd_store(vy0, y);
        simd_store(vy1, y + 16);
        simd_store(vy2, y + 32);
        simd_store(vy3, y + 48);
        simd_store(vy4, y + 64);
        simd_store(vy5, y + 80);
        simd_store(vy6, y + 96);
        simd_store(vy7, y + 112);


        // 更新指针
        x += 128;
        y += 128;
    }


    // 中层循环，处理长度小于 128 但大于或等于 64 的剩余数据
    // 每次以 64 个元素为一组进行处理
    // 每次循环处理完一组后，指针移动 64 个元素位置继续处理下一组
    for (; i <= len - 64; i += 64) {
        simd_load(vx0, x);
        simd_load(vx1, x + 16);
        simd_load(vx2, x + 32);
        simd_load(vx3, x + 48);


        vy0 = simd_cvt_f2h(vx0);
        vy1 = simd_cvt_f2h(vx1);
        vy2 = simd_cvt_f2h(vx2);
        vy3 = simd_cvt_f2h(vx3);


        simd_store(vy0, y);
        simd_store(vy1, y + 16);
        simd_store(vy2, y + 32);
        simd_store(vy3, y + 48);


        x += 64;
        y += 64;
    }


    // 内层循环，处理长度小于 64 但大于或等于 16 的剩余数据
    // 每次以 16 个元素为一组进行处理
    for (; i <= len - 16; i += 16) {
        simd_load(vx0, x);
        vy0 = simd_cvt_f2h(vx0);
        simd_store(vy0, y);


        x += 16;
        y += 16;
    }


    // 最内层循环，处理长度小于 16 的剩余数据
    // 逐元素进行赋值操作
    for (; i < len; i++) {
        *y = *x;
        x++;
        y++;
    }
}


// 根据长度和对齐情况调用 align_batch_function 进行浮点数到半精度浮点数的转换
__device__ void batch_S2H(float* x, half* y, int len)
{
    if (len < 32) {
        for (int i = 0; i < len; i++) {
            y[i] = x[i];
        }
        return;
    }
    const int align_n1 = sizeof(floatv16);
    const int align_n2 = sizeof(halfv16);
    if ((((size_t)x & (align_n1 - 1)) == 0) && ((((size_t)y) & (align_n2 - 1)) == 0)) {
        align_batch_function((float*)x, y, len);
        return;
    } else {
        int padding_x = ((size_t)(x) & (align_n1 - 1)) / sizeof(float);
        int padding_y = ((size_t)(y) & (align_n2 - 1)) / sizeof(half);
        if (padding_x == padding_y) {
            for (int i = 0; i < 16 - padding_x; i++) {
                y[i] = x[i];
            }
            align_batch_function(x + 16 - padding_x, y + 16 - padding_y, len - 16 + padding_x);
            return;
        } else {
            align_batch_function((float*)x, y, len);
            return;
        }
    }
}


__global__ void tecoKernelGemmDoubleBufferImpl(GEMMArgs pGemm)
{
    // 划分线程的行和列
    const int rid = threadIdx / 8;
    const int cid = threadIdx % 8;


    // 从 GEMMArgs 结构体中获取矩阵的维度信息
    const int M = pGemm.m;
    const int N = pGemm.n;
    const int K = pGemm.k;


    //   从 GEMMArgs 结构体中获取矩阵的存储步长
    const int lda = pGemm.lda;
    const int ldb = pGemm.ldb;
    const int ldc = pGemm.ldc;


    // 指向输入矩阵 A、B 和输出矩阵 C 的指针
    const half *A = (const half *)pGemm.A;
    const half *B = (const half *)pGemm.B;
    half *C = (half *)pGemm.C;


    // 外部块的大小
    const int bM = pGemm.bM;
    const int bN = pGemm.bN;
    const int bK = pGemm.bK;


    // 外部块的数量
    int nM = M / bM;
    int nN = N / bN;
    int nK = K / bK;


    // 内部块的大小
    int LocalbM = bM / 8;
    int LocalbN = bN / 4;
    int LocalbK = bK;


    // 内部块的位置
    const half *pA = A + cid * LocalbM * lda;
    const half *pB = B + rid * LocalbN;
    half *pC = C + cid * LocalbM * ldc + rid * LocalbN;


    // A、B、C 的当前块和下一个块的指针
    const half *pCurrA, *pNextA;
    const half *pCurrB, *pNextB;
    half *pCurrC, *pNextC;


    // 计算内存步长
    const int LenA = LocalbM * LocalbK * sizeof(half);
    const int BsizeA = LocalbK * sizeof(half);
    const int StrideA = (lda - LocalbK) * sizeof(half);


    const int LenB = LocalbK * LocalbN * sizeof(half);
    const int BsizeB = LocalbN * sizeof(half);
    const int StrideB = (ldb - LocalbN) * sizeof(half);


    const int LenC = LocalbM * LocalbN * sizeof(half);
    const int BsizeC = LocalbN * sizeof(half);
    const int StrideC = (ldc - LocalbN) * sizeof(half);


    int spm_size = 2 * LenA + 3 * LenB + LenC + LenC / sizeof(half) * sizeof(float) * 2;


    // 为 A、B 的子块和结果 C 分配SPM缓冲区，包括双缓冲区
    half *LocalA = (half *)malloc(LenA * 2);
    half *LocalCompA = LocalA + LocalbM * LocalbK;
    half *LocalB = (half *)malloc(LenB * 3);
    half *LocalCompB = LocalB + LocalbK * LocalbN;
    half *LocalTB = LocalB + LocalbK * LocalbN * 2;
    half *LocalC = (half *)malloc(LenC);
    float *LocalCompC = (float *)malloc(LocalbM * LocalbN * sizeof(float) * 2);
    float *tempC = LocalCompC + LocalbM * LocalbN;


    // 中间计算的临时变量
    half *pTemp;


    // 矩阵A、B 访存和计算缓冲区的指针
    half *pDA = LocalA, *pCA = LocalCompA;
    half *pDB = LocalB, *pCB = LocalCompB;


    int typeAThreads = (3 - cid % 4) * 8 + cid;   // 列广播
    int typeBThreads = rid * 8 + rid;             // 行广播


    // 遍历块和块内元素的循环变量
    int im, in, ik;
    int index_M, index_N, index_K;


    // B 置换的准备
    floatv16 vtb[32];


    // 开始性能采样
    perf_start();


    // 初始化矩阵乘法操作
    MatmulHandle handleMMA;
    matmul_init(handleMMA, MatmulHalfToFloat);


    // 广播的准备
    uint64_t mask = 0;
    mask = 0x01010101 << cid;
    ThreadGroup col_thread_group(mask);


    mask = 0xFF << (8 * rid);
    ThreadGroup row_thread_group(mask);


    // 创建广播句柄
    BroadcastHandle col_handle(&col_thread_group);
    BroadcastHandle row_handle(&row_thread_group);
    sync_threads();


    // 非阻塞广播矩阵 A 和 B
    broadcast_async(pDA, (void *)pA, LenA, BsizeA, {0, (size_t)StrideA}, typeAThreads, BroadcastGlobalToSpm, col_handle);
    broadcast_async(pDB, (void *)pB, LenB, BsizeB, {0, (size_t)StrideB}, typeBThreads, BroadcastGlobalToSpm, row_handle);


    // 遍历矩阵 A 和 B 的块，计算块积并累加到矩阵 C 中
    for (int idx = 0; idx < nM * nN; ++idx) {
        // 计算 M、N 维度的块索引
        index_M = idx / nN;
        index_N = idx % nN;
        // 遍历 K 维度，将 A 和 B 的块相乘并将结果累加到 TempC 中
        for (index_K = 0; index_K < nK; ++index_K) {
            // 更新矩阵 A 和 B 的当前块的指针
            pCurrA = pA + index_M * bM * lda + index_K * bK;     // A[index_M][index_K]
            pCurrB = pB + index_K * bK * ldb + index_N * bN;     // B[index_K][index_N]
            // 定位 pNextA 和 pNextB
            if (index_K < nK - 1) {
                pNextA = pA + index_M * bM * lda + (index_K + 1) * bK;       // A[index_M][index_K+1]
                pNextB = pB + (index_K + 1) * bK * ldb + index_N * bN;       // B[index_K+1][index_N]
            } else {
                if (index_N == nN - 1) {
                    pNextA = pA + (index_M + 1) * bM * lda;   // A[index_M+1][0]
                    pNextB = pB;                         // B[0][0]
                    if (idx == (nM * nN - 1)) pNextA = pA;
                } else {
                    pNextA = pA + index_M * bM * lda;   // A[index_M][0]
                    pNextB = pB + (index_N + 1) * bN;   // B[0][index_N+1]
                }
            }


            // 交换 pCA 和 pDA
            pTemp = pCA;
            pCA = pDA;
            pDA = pTemp;


            // 交换 pCB 和 pDB
            pTemp = pCB;
            pCB = pDB;
            pDB = pTemp;


            // 等待 A 广播和矩阵乘法数据加载
            broadcast_wait(col_handle);
            matmul_wait_loading_input(handleMMA);


            // 同步列线程组内的线程
            sync_threads(col_thread_group);


            // 非阻塞广播矩阵 A 的下一个块
            broadcast_async(pDA, (void *)pNextA, LenA, BsizeA, {0, (size_t)StrideA}, typeAThreads, BroadcastGlobalToSpm,
                            col_handle);


            // 对矩阵 B 进行相同操作
            broadcast_wait(row_handle);
            matmul_wait_loading_weight(handleMMA);


            sync_threads(row_thread_group);


            broadcast_async(pDB, (void *)pNextB, LenB, BsizeB, {0, (size_t)StrideB}, typeBThreads, BroadcastGlobalToSpm,
                            row_handle);


            // 置换矩阵 B
            for (ik = 0; ik < LocalbK; ik += 8) {
                simd_load(vtb[0], (float *)(pCB + ik * LocalbN));
                simd_load(vtb[1], (float *)(pCB + ik * LocalbN + 32));
                simd_load(vtb[2], (float *)(pCB + (ik + 1) * LocalbN));
                simd_load(vtb[3], (float *)(pCB + (ik + 1) * LocalbN + 32));
                simd_load(vtb[4], (float *)(pCB + (ik + 2) * LocalbN));
                simd_load(vtb[5], (float *)(pCB + (ik + 2) * LocalbN + 32));
                simd_load(vtb[6], (float *)(pCB + (ik + 3) * LocalbN));
                simd_load(vtb[7], (float *)(pCB + (ik + 3) * LocalbN + 32));
                simd_load(vtb[8], (float *)(pCB + (ik + 4) * LocalbN));
                simd_load(vtb[9], (float *)(pCB + (ik + 4) * LocalbN + 32));
                simd_load(vtb[10], (float *)(pCB + (ik + 5) * LocalbN));
                simd_load(vtb[11], (float *)(pCB + (ik + 5) * LocalbN + 32));
                simd_load(vtb[12], (float *)(pCB + (ik + 6) * LocalbN));
                simd_load(vtb[13], (float *)(pCB + (ik + 6) * LocalbN + 32));
                simd_load(vtb[14], (float *)(pCB + (ik + 7) * LocalbN));
                simd_load(vtb[15], (float *)(pCB + (ik + 7) * LocalbN + 32));
                simd_store(vtb[0], (float *)(LocalTB + ik * 32));
                simd_store(vtb[1], (float *)(LocalTB + 32 * LocalbK + ik * 32));
                simd_store(vtb[2], (float *)(LocalTB + (ik + 1) * 32));
                simd_store(vtb[3], (float *)(LocalTB + 32 * LocalbK + (ik + 1) * 32));
                simd_store(vtb[4], (float *)(LocalTB + (ik + 2) * 32));
                simd_store(vtb[5], (float *)(LocalTB + 32 * LocalbK + (ik + 2) * 32));
                simd_store(vtb[6], (float *)(LocalTB + (ik + 3) * 32));
                simd_store(vtb[7], (float *)(LocalTB + 32 * LocalbK + (ik + 3) * 32));
                simd_store(vtb[8], (float *)(LocalTB + (ik + 4) * 32));
                simd_store(vtb[9], (float *)(LocalTB + 32 * LocalbK + (ik + 4) * 32));
                simd_store(vtb[10], (float *)(LocalTB + (ik + 5) * 32));
                simd_store(vtb[11], (float *)(LocalTB + 32 * LocalbK + (ik + 5) * 32));
                simd_store(vtb[12], (float *)(LocalTB + (ik + 6) * 32));

                simd_store(vtb[14], (float *)(LocalTB + (ik + 7) * 32));
                simd_store(vtb[15], (float *)(LocalTB + 32 * LocalbK + (ik + 7) * 32));

            if (index_K < nK - 1) {
                for (ik = 0; ik + 32 - 1 < LocalbK; ik += 32) {

                    matmul_set_flushing_output(handleMMA, false);
                    matmul_set_output_row_offset(handleMMA, MatmulEnableOutputRowOffset, 0);
                    matmul_load_weight(handleMMA, LocalTB + ik * LocalbN / 2, MatmulK32, MatmulN32);

                                   MatmulK32, MatmulN32);
                matmul_compute(handleMMA, pCA + ik, LocalbM, MatmulK32, LocalbK / REDUITEM - 1);
                matmul_store(handleMMA, LocalCompC, LocalbM * 2, MatmulN32);

        matmul_wait(handleMMA);

        for (im = 0; im < LocalbM; im++) {

            simd_load(vtb[1], LocalCompC + im * 32 + 16);
            simd_load(vtb[2], LocalCompC + (im + LocalbM) * 32);

    perf_stop();

    perf_print();
```

## 17. 常见问题

### 17.1 如何使用 CMake 构建？

为使 CMake 可以构建后缀为 `.scpp` 的文件，需要先使用 `set_source_files_properties` 命令，设置 `LINKER_LANGUAGE` 的属性为 `CXX`。CMake 构建分为 LTO 模式和非 LTO 模式。

- **LTO 模式**：在生成动态库或者可执行文件时，用到的所有 `.scpp` 文件都在同一个编译模块里，即：一个 `.scpp` 里定义的变量和函数，可以在另一个 `.scpp` 文件里使用。
- **非 LTO 模式**：不同的 `.scpp` 文件是不同的编译模块。在不同编译模块中的设备端代码，相互之间没有依赖关系，即：在一个编译模块里定义的变量和函数，在另一个编译模块里不能对其进行访问。
