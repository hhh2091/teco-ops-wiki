## 15.2.3（续）设备端矩阵乘计算与核函数

本节延续基于矩阵乘计算单元的 SPMD 矩阵乘示例。下面的代码片段展示了矩阵乘计算的双缓冲流水（加载后 32 行权重并与输入矩阵右半相乘）、核函数 `matmul_func` 的 SPM 申请与数据搬运，以及主机端 `main` 函数的完整驱动逻辑。

### 矩阵乘计算流水（接续片段）

```c
     // 使能下一次计算为最后一次计算
     // 计算完后会输出累加缓冲区内的数据
     matmul_set_flushing_output(handle, true);


     // 将权重矩阵后32行加载到矩阵乘计算单元上
     matmul_load_weight(handle, weight + MatmulK32 * MatmulN32, MatmulK32, MatmulN32);
     // 等待权重矩阵数据加载完成，此后可覆写weight上后32行数据
     matmul_wait_loading_weight(handle);


     // 跨步加载输入矩阵（此处相当于加载输入矩阵右半）
     // 并计算与权重矩阵的乘积
     matmul_compute(handle, input + MatmulK32, m, MatmulK32, stride);
     // 等待输入矩阵加载完成，此后可覆写input上右半数据
     matmul_wait_loading_input(handle);


     // 将矩阵乘运算结果写回到SPM指定位置
     matmul_store(handle, output, m, MatmulN32);


     // 等待运算结果写回完成
     matmul_wait(handle);
}
```

### 核函数 `matmul_func`

```c
__global__ void matmul_func(char *data_input, char *data_weight, char *data_output, int m, int k, int n)
{
     // 计算每个计算核心SPE所需要承担的计算任务
     unsigned int slice = m / threadDim;


     // 申请SPM存储空间
     short *input = (short*)malloc(slice * k * sizeof(short));
     short *weight = (short*)malloc(k * n * sizeof(short));
     short *output = (short*)malloc(slice * n * sizeof(short));


     // 将数据从Global搬运输入矩阵和权重矩阵到SPM
     memcpy_async(input, data_input + threadIdx * slice * k * sizeof(short), slice * k * sizeof(short),
 0, {0, 0}, MemcpyGlobalToSpm);
     memcpy_async(weight, data_weight, k * n * sizeof(short), 0, {0, 0}, MemcpyGlobalToSpm);
     // 等待异步拷贝完成
     memcpy_wait();


     // 矩阵乘计算
     matmul_compute_func(input, weight, output, slice, k, n);


     // 将输出矩阵从SPM搬运回Global
     memcpy(data_output + threadIdx * slice * n * sizeof(short), output, slice * n * sizeof(short));


     // 释放在SPM申请的内存
     free(output);
     free(weight);
     free(input);
}
```

### 主机端 `main` 函数

```c
int main()
{
    // 分配Host端空间
    // h_input为输入矩阵，h_weight为权重矩阵，h_output为计算结果矩阵
    // h_output_reference为Host模仿矩阵乘计算，计算结果存储矩阵
    short* h_input = (short*)malloc(M * K * sizeof(short));
    short* h_weight = (short*)malloc(K * N * sizeof(short));
    short* h_output = (short*)malloc(M * N * sizeof(short));
    short* h_output_reference = (short*)malloc(M * N * sizeof(short));


    srand(time(NULL));
    // 为Host端输入矩阵进行初始化
    for (int i = 0; i < M; i++) {
        for (int j = 0; j < K; j++) {
             h_input[i * K + j] = rand() % 10;
        }
    }


    // 为Host端权重矩阵进行初始化
    for (int i = 0; i < K; i++) {
        for (int j = 0; j < N; j++) {
             h_weight[i * N + j] = rand() % 5;
        }
    }


    // 在Host模拟矩阵乘操作，并将结果存放到h_output_reference
    memset(h_output_reference, 0, M * N * sizeof(short));
    for (int i = 0; i < M; i++) {
        for (int j = 0; j < N; j++) {
             for (int k = 0; k < K; k++) {
                   h_output_reference[i * N + j] +=
                      h_input[i * K + k] * h_weight[k * N + j];
             }
        }
    }


    // 配置使用0号计算核心阵列
    sdaaSetDevice(0);
    // 声明Device端地址
    // dev_input为输入矩阵，dev_weight为权重矩阵，dev_output为计算结果矩阵
    char* dev_input = NULL;
    char* dev_weight = NULL;
    char* dev_output = NULL;


    // 使用sdaaMalloc为Device端地址进行空间分配
    sdaaMalloc((void**)&dev_input, M * K * sizeof(short));
    sdaaMalloc((void**)&dev_weight, K * N * sizeof(short));
    sdaaMalloc((void**)&dev_output, M * N * sizeof(short));


    // 使用sdaaMemcpy将Host端内存拷贝到Device端
    sdaaMemcpy(dev_input, h_input, M * K * sizeof(short), sdaaMemcpyHostToDevice);
    sdaaMemcpy(dev_weight, h_weight, K * N * sizeof(short), sdaaMemcpyHostToDevice);


    // 启动核函数调用
    matmul_func<<<1>>>(dev_input, dev_weight, dev_output, M, K, N);


    // 等待计算核心SPE完成之前的所有请求任务
    sdaaDeviceSynchronize();


    // 使用sdaaMemcpy将Device端矩阵乘计算结果拷贝到Host端
    sdaaMemcpy(h_output, dev_output, M * N * sizeof(short), sdaaMemcpyDeviceToHost);


    // 将Device端矩阵乘计算结果与Host端计算结果进行对比
    // 验证失败则打印对应日志
    for (int i = 0; i < M * N; i++) {
            if(h_output[i] != h_output_reference[i]) {
            printf("find mismatching data on line %d: ace: %d, serial: %d\n",
                      i, (int)h_output[i], (int)h_output_reference[i]);
            break;
        }


        // 如果验证成功，则打印对应日志
        if (i == M * N - 1) {
            printf("matmul result compare ok\n");
        }
    }


    // 释放在Device端和Host申请的内存
    free(h_output_reference);
    free(h_output);
    free(h_weight);
    free(h_input);
    sdaaFree(dev_output);
    sdaaFree(dev_weight);
    sdaaFree(dev_input);


    return 0;
}
```

## 15.3 使用SUMMA算法实现矩阵乘

### 15.3.1 概述

本例将在太初AI加速卡上通过使用SUMMA算法实现矩阵乘操作。完整程序请参考完整程序。

主机端通过SDAARuntime接口分配设备端的Global存储空间，并将数据从主机端拷贝到设备端的Global存储空间。在设备端使用SDAA C提供的DMA数据搬运接口、Broadcast数据搬运接口、矩阵乘接口、SIMD接口等完成SUMMA算法，实现对大矩阵的矩阵乘运算。设备端完成计算后通过SDAARuntime接口将Global存储空间的计算结果拷贝到主机端内存。

### 15.3.2 编程思路

#### 主机端

1. 分配主机端内存并通过随机数进行初始化操作。
2. 主机端与设备端的数据传输。
   a. 设置计算核心阵列SPA号。
   b. 分配Global存储空间。
   c. 从主机端内存向Global存储空间传输数据。
   d. 调用Kernel函数。
   e. 将计算结果从Global存储空间传回主机端内存。
3. 在主机端模拟矩阵乘运算，并与设备端计算结果进行对比。
4. 释放Global存储空间与主机端内存。

#### 设备端

1. 将计算核心阵列SPA内的SPE按照4行8列的方式抽象为一个二维数组。
2. 将输入矩阵与权重矩阵按照抽象的行列数量切分为若干个子矩阵。
3. 在SPE内为子矩阵申请SPM存储空间。
4. 通过阻塞型DMA接口memcpy将小矩阵拷贝到申请的SPM存储空间内。
5. 在K维度上循环并通过阻塞型Broadcast接口对小矩阵根据抽象的行列按行、按列进行广播操作。
6. 每个SPE在接收到广播数据后进行矩阵乘运算。
7. 将本次的矩阵乘结果上一次的矩阵乘计算结果进行累加操作。
8. 将单个SPE的计算结果通过阻塞型DMA数据搬运接口拷贝到Global存储空间。
9. 释放申请的SPM存储空间。

### 15.3.3 编写程序

#### 15.3.3.1 主机端

##### 主机端程序设计思路

1. 分配主机端内存并通过随机数进行初始化操作
   a. 在主机端通过 `malloc` 接口分配3个主机端存储空间 `h_input`、`h_weight`、`h_output`。
      - `h_input`：输入矩阵。
      - `h_weight`：权重矩阵。
      - `h_output`：输出矩阵。
   b. 通过随机数将输入矩阵与权重矩阵进行初始化操作。
2. 主机端与设备端的数据传输
   a. 设置计算核心阵列SPA号：通过 `sdaaSetDevice` 接口指定Kernel函数运行在哪一个计算核心阵列上。
   b. 分配Global存储空间：通过 `sdaaMalloc` 接口在设备端分配3个Global存储空间（`dev_input`、`dev_weight`、`dev_output`）。
      - `dev_input`：Global存储空间内的输入矩阵。
      - `dev_weight`：Global存储空间内的权重矩阵。
      - `dev_output`：Global存储空间内的输出矩阵。
   c. 数据传输：通过 `sdaaMemcpy` 接口将Host端内存（`h_input`、`h_weight`、`h_output`）对应拷贝到Device端Global存储空间（`dev_input`、`dev_weight`、`dev_output`）。
   d. 调用Kernel函数：通过 `<<<…>>>` 调用Kernel函数。
   e. 拷贝计算结果：通过 `sdaaDeviceSynchronize` 接口等待计算核心SPE完成之前的所有请求任务，再通过 `sdaaMemcpy` 接口将Device端Global存储空间 `dev_output` 中的数据拷贝到Host内存 `h_output`。
3. 计算结果对比与内存输出
   a. 在Host端模拟矩阵乘计算，并将计算结果存放到 `h_output_reference`。
   b. 计算结果对比：将Kernel的计算结果 `h_output` 与参考矩阵乘计算结果 `h_output_reference` 进行对比。
   c. 释放内存：
      i. 释放申请的Device端Global存储空间：通过 `sdaaFree` 接口释放Device端的Global存储空间。
      ii. 释放申请的Host端存储空间：通过 `free` 接口释放申请的Host端存储空间。

##### 主机端程序

```c
#include <stdio.h>


// 定义SUMMA算法的矩阵规模
#define M 64 // 输入矩阵行数
#define K 32 // 输入矩阵最大列数，权重矩阵最大行数
#define N 64 // 权重矩阵最大列数


int main()
{
    int i = 0, j = 0, k = 0;
    // 分配Host端地址
    // h_input为输入矩阵，h_weight为权重矩阵，h_output为计算结果矩阵
    half *h_input = (half *)malloc(M * K * sizeof(half));
    half *h_weight = (half *)malloc(K * N * sizeof(half));
    half *h_output = (half *)malloc(M * N * sizeof(half));


    // 设置随机种子
    srand(100);


    // 为Host端输入矩阵进行初始化
    memset(h_input, 0, M * K * sizeof(half));
    for (i = 0; i < M; i++) {
         for (j = 0; j < K; j++) {
              h_input[i * K + j] = 1.0 * rand() / RAND_MAX * 10.0 - 5;
         }
    }
    // 为Host端权重矩阵进行初始化
    memset(h_weight, 0, K * N * sizeof(half));
    for (i = 0; i < K; i++) {
         for (j = 0; j < N; j++) {
              h_weight[i * N + j] = 1.0 * rand() / RAND_MAX * 0.1 - 0.05;
         }
    }


    // 配置使用0号计算核心阵列
    sdaaSetDevice(0);
    // 声明Device端地址
    // dev_input为输入矩阵，dev_weight为权重矩阵，dev_output为计算结果矩阵
    char *dev_input = NULL;
    char *dev_weight = NULL;
    char *dev_output = NULL;


    sdaaMalloc((void **)&dev_input, M * K * sizeof(half));
    sdaaMalloc((void **)&dev_weight, K * N * sizeof(half));
    sdaaMalloc((void **)&dev_output, M * N * sizeof(half));


    // 使用sdaaMemcpy将Host端内存拷贝到Device端
    sdaaMemcpy(dev_input, h_input, M * K * sizeof(half), sdaaMemcpyHostToDevice);
    sdaaMemcpy(dev_weight, h_weight, K * N * sizeof(half), sdaaMemcpyHostToDevice);


    // 声明Device端dev_outputheck_val
    unsigned int *dev_outputheck_val = NULL;
    // 使用sdaaMalloc为dev_outputheck_val进行空间分配
    sdaaMalloc((void **)&dev_outputheck_val, threadDim * sizeof(unsigned int));
    // 为check_val在host端进行空间分配
    unsigned int *check_val = (unsigned int *)malloc(threadDim * sizeof(unsigned int));


    // 核函数调用
    summa<<<1>>>(dev_input, dev_weight, dev_output, dev_outputheck_val, M, K, N);


    // 等待计算核心SPE完成之前的所有请求任务
    sdaaDeviceSynchronize();


    // 使用sdaaMemcpy将Device端矩阵乘计算结果拷贝到Host端
    sdaaMemcpy(h_output, dev_output, M * N * sizeof(half), sdaaMemcpyDeviceToHost);


    // 使用sdaaMemcpy将Device端验证输入参数合法性结果拷贝到Host端
    sdaaMemcpy(check_val, dev_outputheck_val, threadDim * sizeof(unsigned int), sdaaMemcpyDeviceToHost);


    // 对check结果进行合法性校验，验证失败则打印验证失败日志，否则打印验证成功日志
    bool check_result = true;
    for (i = 0; i < threadDim; i++) {
        if (check_val[i] != 0) {
            check_result = false;
            printf("thread %d check_matmul failed, returns %d\n", i, *check_val);
        }
    }


    if (check_result) {
        printf("all check_matmul checked done\n");


        // 在Host模拟矩阵乘操作，并将结果存放到h_output_reference
        half *h_output_reference = (half *)malloc(M * N * sizeof(half));
        memset(h_output_reference, 0, M * N * sizeof(half));
        for (i = 0; i < M; i++) {
            for (j = 0; j < N; j++) {
                for (k = 0; k < K; k++) {
                    h_output_reference[i * N + j] += h_input[i * K + k] * h_weight[k * N + j];
                }
            }
        }


        // 将Device端矩阵乘计算结果与Host端计算结果进行对比，验证失败则打印对应日志
        for (i = 0; i < M * N; i++) {
            if ((h_output[i] - h_output_reference[i] > 1.5e-2)) {
                printf("find mismatching data Pos %d, on line %d, col %d: ace: %f, serial: %f\n",
                         i, i / N, i % N, (float)h_output[i], (float)h_output_reference[i]);
                break;
            }
            if (i == M * N - 1) {
                printf("SUMMA result compare ok\n");
            }
        }


        // 释放h_output_reference空间
        free(h_output_reference);
    }


    // 释放在Device端和Host申请的内存
    free(check_val);
    sdaaFree(dev_outputheck_val);
    sdaaFree(dev_output);
    sdaaFree(dev_weight);
    sdaaFree(dev_input);
    free(h_output);
    free(h_weight);
    free(h_input);


    return 0;
}
```

#### 15.3.3.2 设备端

##### 设备端程序设计思路

设备端Kernel函数通过SUMMA算法将输入矩阵切分为若干个子矩阵，计算核心SPE将相应的子矩阵加载到SPM存储空间后，通过SDAA C提供的Broadcast数据搬运接口、矩阵乘接口、SIMD接口等完成子矩阵的计算。最后将子矩阵的计算结果进行合并完成整体计算任务的求解。

> **说明：**
>
> 1. 使用 `__device__` 关键字修饰，说明该变量或函数是在设备端的Global存储空间，主机端不允许访问。
> 2. 使用 `__global__` 关键字修饰，说明函数是Kernel函数，允许主机端调用，但不允许设备端调用。返回值类型仅为void类型。
> 3. `threadDim` 用于获取计算核心阵列可用的计算核心资源数量。
> 4. `threadIdx` 用于读取计算核心SPE的ID。

1. 将计算核心阵列SPA内的SPE按照4行8列的方式抽象为一个二维数组。
2. 将输入矩阵与权重矩阵按照抽象的行列数量切分为若干个子矩阵（每个计算核心阵列SPE对应一个子矩阵）。
3. 在计算核心SPE内为子矩阵申请SPM存储空间（`input_spm`、`weight_spm`、`output_spm`）。
   - `input_spm`：输入子矩阵。
   - `weight_spm`：权重子矩阵。
   - `output_spm`：输出子矩阵。
4. 通过阻塞型DMA数据搬运接口将子矩阵数据从Global存储空间拷贝到SPM存储空间。
5. 计算广播的行列掩码，并初始化BroadcastHandle类型的广播句柄。
6. 为广播地址申请SPM存储空间。
   - `input_dst_spm`：输入子矩阵多播的目的空间。
   - `weight_dst_spm`：权重子矩阵多播的目的空间。
   - `input_src_spm`：输入子矩阵多播的原始数据空间。
   - `weight_dst_spm`：权重子矩阵多播的原始数据空间。
7. 在K维度上对输入子矩阵和权重子矩阵进行循环操作。
   a. 将输入子矩阵的列广播到与当前计算核心SPE同行的计算核心SPE申请的 `input_dst_spm` 存储空间内。
   b. 将权重子矩阵的列广播到与当前计算核心SPE同列的计算核心SPE申请的 `weight_dst_spm` 存储空间内。
   c. 将 `input_dst_spm` 存储空间与 `weight_dst_spm` 存储空间内的数据通过阻塞型矩阵乘接口进行矩阵乘计算。
   d. 通过SIMD接口对每次的矩阵乘计算结果进行累加操作。
8. 计算核心SPE完成子矩阵计算后将计算结果通过阻塞型DMA数据搬运接口拷贝到Global存储空间内。
9. 释放申请的SPM存储空间。

##### 设备端程序

```cpp
#include <sdaa_matmul.h>
using namespace sdaa;


// 预定义SUMMA算法使用的在遍历K维时的步长
#define NB 2


// 定义当前SPA的阵列
#define MESH_ROW_NUM 4 // 行数
#define MESH_COL_NUM 8 // 列数


// 定义行多播核列多播时的枚举标记
enum Mask_Flag {
      MASK_COL = 0,
      MASK_ROW = 1
};


// 矩阵乘法的封装函数，包含参数合法性校验
__device__ unsigned int matmul_wrapper(const char *input, const char *weight, char *output, unsigned int *check_val,
                                       const int dev_m, const int dev_k, const int dev_n)
{
      // 合法性校验
      *check_val += check_matmul(MatmulHalfToHalf, output, input, weight,
                                   dev_m, dev_k, dev_n);
      if (*check_val != MATMUL_SUCC) {
          return *check_val;
      }


      // 执行矩阵乘法
      matmul(MatmulHalfToHalf, output, input, weight, dev_m, dev_k, dev_n);


      return 0;
}


// 计算多播时需要的mask参数，主要针对SUMMA算法中的水平方向多播和垂直方向多播
__device__ unsigned long gen_broadcast_mask(Mask_Flag is_row)
{
    // 计算当前线程所在SPA内的坐标
    const int x_pos = threadIdx % MESH_COL_NUM;
    const int y_pos = threadIdx / MESH_COL_NUM;


    unsigned long mask = 0;
    // 计算列多播时的mask
    if (is_row == MASK_COL) {
        int start_id = x_pos;
        for (int i = 0; i < threadDim; i += MESH_COL_NUM) {
            mask |= (1 << (start_id + i));
        }
    }
    // 计算行多播时的mask
    if (is_row == MASK_ROW) {
        int start_id = y_pos * MESH_COL_NUM;
        for (int i = 0; i < MESH_COL_NUM; i++) {
            mask |= (1 << (start_id + i));
        }
    }
    return mask;
}


__device__ void load_sub_block(char *src, char *dst, const int src_h, const int src_w, const int dst_h,
                                 const int dst_w, const int start_x, const int start_y)
{
    const int type_size = sizeof(half);
    // 计算原数据在copy时的步长
    const int src_stride = src_w * type_size;
    // 计算原数据在copy时的起始地址
    char *src_start = src + (start_y * src_w + start_x) * type_size;
    for (int i = 0; i < dst_h; i++) {
        memcpy(dst, src_start, dst_w * type_size);
        src_start += src_stride;
        dst += dst_w * type_size;
    }
}


__device__ void gen_sub_block(char *src, char *dst, const int src_h, const int src_w, const int dst_h,
                                const int dst_w, const int start_x, const int start_y)
{
    const int type_size = sizeof(half);
    // 计算原数据在copy时的起始地址
    char *src_start = src + (start_y * src_w + start_x) * type_size;
    // 计算原数据在copy时Global存储空间的步长
    const size_t glbmem_stride = (src_w - dst_w) * type_size;


    memcpy_stride(dst, src_start, dst_w * type_size * dst_h, dst_w * type_size, {0, glbmem_stride});
}


__device__ void save_sub_block(char *dst_glb, char *src_spm, const int dst_h, const int dst_w,
                               const int src_h, const int src_w, const int start_x, const int start_y)
{
    const int type_size = sizeof(half);
    // 计算目标数据在copy时的步长
    const int dst_stride = dst_w * type_size;
    // 计算目标数据在copy时的起始地址
    char *dst_start = dst_glb + (start_y * dst_w + start_x) * type_size;
    for (int i = 0; i < src_h; i++) {
        memcpy(dst_start, src_spm, src_w * type_size);
        dst_start += dst_stride;
        src_spm += src_w * type_size;
    }
}


// summa算法的核函数
__global__ void summa(char *input, char *weight, char *output, unsigned int *check_val,
                      int dev_m, int dev_k, int dev_n)
{
    // 计算数据类型大小
    const int type_size = sizeof(half);
    // 计算当前线程所在SPA内的坐标
    const int x_pos = threadIdx % MESH_COL_NUM;
    const int y_pos = threadIdx / MESH_COL_NUM;


    // 计算输入矩阵分块后子矩阵的行数
    const int input_sub_m = dev_m / MESH_ROW_NUM;
    // 计算输入矩阵分块后子矩阵的列数
    const int input_sub_k = dev_k / MESH_COL_NUM;
    // 计算权重矩阵分块后子矩阵的行数
    const int weight_sub_k = dev_k / MESH_ROW_NUM;
    // 计算权重矩阵分块后子矩阵的列数
    const int weight_sub_n = dev_n / MESH_COL_NUM;
    // 计算输入矩阵子阵的大小
    const int input_spm_num = input_sub_m * input_sub_k;
    // 计算权重矩阵子矩阵的大小
    const int weight_spm_num = weight_sub_k * weight_sub_n;
    // 计算输出矩阵子矩阵的大小
    const int output_spm_num = input_sub_m * weight_sub_n;


    // 输入子矩阵多播的数据量
    const int input_broad_num = NB * input_sub_m;
    // 权重子矩阵多播的数据量
    const int output_broad_num = NB * weight_sub_n;


    // 判断每次多播数据是否合法，必须是4字节的倍数
    if (((input_broad_num & 1) != 0) || ((output_broad_num & 1) != 0)) {
        printf("the broadcast data number must be the times of 4 bytes is not meet!");
        return;
    }


    // 在SPM上申请内存
    // 输入矩阵分块后的子矩阵空间
    char *input_spm = (char *)malloc(input_spm_num * type_size);
    // 权重矩阵分块后的子矩阵空间
    char *weight_spm = (char *)malloc(weight_spm_num * type_size);
    // 输出矩阵分块后的子矩阵空间
    char *output_spm = (char *)malloc(output_spm_num * type_size);
    // 输入矩阵和权重矩阵多播后每次临时矩阵计算后的结果空间
    char *output_rst_spm = (char *)malloc(output_spm_num * type_size);


    // 初始化output_spm为0
    memset(output_spm, 0, output_spm_num * type_size);
    // 初始化check_val为0
    memset(check_val, 0, threadDim * sizeof(unsigned int));


    // 从Global加载输入矩阵的子矩阵到SPM
    gen_sub_block(input, input_spm, dev_m, dev_k, input_sub_m, input_sub_k,
                    input_sub_k * x_pos, input_sub_m * y_pos);
    // 从Global加载权重矩阵的子矩阵到SPM
    gen_sub_block(weight, weight_spm, dev_k, dev_n, weight_sub_k, weight_sub_n,
                    weight_sub_n * x_pos, weight_sub_k * y_pos);


    // 计算多播相关参数
    unsigned long row_mask = gen_broadcast_mask(MASK_ROW);
    unsigned long col_mask = gen_broadcast_mask(MASK_COL);


    ThreadGroup row_thread_group(row_mask);
    ThreadGroup col_thread_group(col_mask);


    // 设置多播mask参数
    BroadcastHandle handle_row(&row_thread_group);
    BroadcastHandle handle_col(&col_thread_group);


    // 广播的源地址和目的地址
    // 输入矩阵分块后的子矩阵多播的目的空间
    char *input_dst_spm = (char *)malloc(input_broad_num * type_size);
    // 权重矩阵分块后的子矩阵多播的目的空间
    char *weight_dst_spm = (char *)malloc(output_broad_num * type_size);
    // 输入矩阵分块后的子矩阵多播的原数据空间
    char *input_src_spm = (char *)malloc(input_broad_num * type_size);
    // 权重矩阵分块后的子矩阵多播的原数据空间
    char *weight_src_spm = (char *)malloc(output_broad_num * type_size);


    // 同一计算核心阵列SPA内所有计算核心SPE同步操作
    sync_threads();


    // 在K维度上进行遍历，每次计算最终结果的一部分
    for (int i = 0; i < dev_k; i += NB) {
        //   计算本次水平广播的发起方ID
        unsigned root_row = (y_pos * MESH_COL_NUM) + (i / input_sub_k);
        // 准备水平广播的数据
        if (root_row == threadIdx) {
             // 由于输入矩阵每次多播的数据为部分列，所以需要
             // 将输入矩阵的多播数据从不连续的子块空间，拷贝到连续的地址空间
             load_sub_block(input_spm, input_src_spm, input_sub_m, input_sub_k, input_sub_m, NB,
                             i % input_sub_k, 0);
        }
        // 调用广播同步接口，输入矩阵进行水平方向广播
        broadcast(input_dst_spm, input_src_spm, input_broad_num * type_size, 0, {0, 0}, root_row, BroadcastSpmToSpm, handle_row);


        // 计算本次垂直广播的发起方ID
        unsigned root_col = x_pos + ((i / weight_sub_k) * MESH_COL_NUM);
        // 准备本次垂直广播的原地址空间数据，因为数据是连续的，调用memcpy一次即可
        memcpy(weight_src_spm, weight_spm + ((i % weight_sub_k) * weight_sub_n * type_size),
                output_broad_num * type_size);
        // 调用广播同步接口，矩阵B进行垂直方向广播
        broadcast(weight_dst_spm, weight_src_spm, output_broad_num * type_size, 0, {0, 0}, root_col, BroadcastSpmToSpm, handle_col);


        // 执行同步操作，确保接收方接收到数据
        sync_threads();


        // 对广播收到的输入矩阵的子矩阵和权重矩阵的子矩阵进行乘法运算
        matmul_wrapper(input_dst_spm, weight_dst_spm, output_rst_spm, check_val + threadIdx,
                          input_sub_m, NB, weight_sub_n);
        // 将每次临时计算的结果进行累加
        halfv16 v_dst, v_src;
        for (int i = 0; i < output_spm_num;) {
            int current_num = ((i + 16) <= output_spm_num) ? 16 : (output_spm_num - i);
            if (16 == current_num) {
                 // 如果当前计算16个连续的数据要求，调用并行指令
                 simd_load(v_dst, output_spm + i * type_size);
                 simd_load(v_src, output_rst_spm + i * type_size);
                 v_dst += v_src;
                 simd_store(v_dst, output_spm + i * type_size);
                 i += 16;
            } else {
                 // 对单个数据，进行串行计算
                 half *src = (half *)output_rst_spm;
                 half *dst = (half *)output_spm;
                 *(dst + i) += *(src + i);
                 i++;
            }
        }
    }
    // 从SPM保存输出矩阵的子矩阵到Global
    save_sub_block(output, output_spm, dev_m, dev_n, input_sub_m, weight_sub_n,
                     weight_sub_n * x_pos, input_sub_m * y_pos);


    // 释放在SPM上申请的内存
    free(input_spm);
    free(weight_spm);
    free(output_spm);
    free(output_rst_spm);
    free(weight_src_spm);
    free(input_src_spm);
    free(weight_dst_spm);
    free(input_dst_spm);
}
```

```cpp
    free(input_src_spm);
    free(weight_dst_spm);
    free(input_dst_spm);
}


int main()
{
    int i = 0, j = 0, k = 0;
    // 分配Host端地址
    // h_input为输入矩阵，h_weight为权重矩阵，h_output为计算结果矩阵
    half *h_input = (half *)malloc(M * K * sizeof(half));
    half *h_weight = (half *)malloc(K * N * sizeof(half));
    half *h_output = (half *)malloc(M * N * sizeof(half));


    // 设置随机种子
    srand(100);


    // 为Host端输入矩阵进行初始化
    memset(h_input, 0, M * K * sizeof(half));
    for (i = 0; i < M; i++) {
        for (j = 0; j < K; j++) {
             h_input[i * K + j] = 1.0 * rand() / RAND_MAX * 10.0 - 5;
        }
    }
    // 为Host端权重矩阵进行初始化
    memset(h_weight, 0, K * N * sizeof(half));
    for (i = 0; i < K; i++) {
        for (j = 0; j < N; j++) {
             h_weight[i * N + j] = 1.0 * rand() / RAND_MAX * 0.1 - 0.05;
        }
    }


    // 配置使用0号计算核心阵列
    sdaaSetDevice(0);
    // 声明Device端地址
    // dev_input为输入矩阵，dev_weight为权重矩阵，dev_output为计算结果矩阵
    char *dev_input = NULL;
    char *dev_weight = NULL;
    char *dev_output = NULL;


    sdaaMalloc((void **)&dev_input, M * K * sizeof(half));
    sdaaMalloc((void **)&dev_weight, K * N * sizeof(half));
    sdaaMalloc((void **)&dev_output, M * N * sizeof(half));


    // 使用sdaaMemcpy将Host端内存拷贝到Device端
    sdaaMemcpy(dev_input, h_input, M * K * sizeof(half), sdaaMemcpyHostToDevice);
    sdaaMemcpy(dev_weight, h_weight, K * N * sizeof(half), sdaaMemcpyHostToDevice);


    // 声明Device端dev_outputheck_val
    unsigned int *dev_outputheck_val = NULL;
    // 使用sdaaMalloc为dev_outputheck_val进行空间分配
    sdaaMalloc((void **)&dev_outputheck_val, threadDim * sizeof(unsigned int));
    // 为check_val在host端进行空间分配
    unsigned int *check_val = (unsigned int *)malloc(threadDim * sizeof(unsigned int));


    // 核函数调用
    summa<<<1>>>(dev_input, dev_weight, dev_output, dev_outputheck_val, M, K, N);


    // 等待计算核心SPE完成之前的所有请求任务
    sdaaDeviceSynchronize();


    // 使用sdaaMemcpy将Device端矩阵乘计算结果拷贝到Host端
    sdaaMemcpy(h_output, dev_output, M * N * sizeof(half), sdaaMemcpyDeviceToHost);


    // 使用sdaaMemcpy将Device端验证输入参数合法性结果拷贝到Host端
    sdaaMemcpy(check_val, dev_outputheck_val, threadDim * sizeof(unsigned int), sdaaMemcpyDeviceToHost);


    // 对check结果进行合法性校验，验证失败则打印验证失败日志，否则打印验证成功日志
    bool check_result = true;
    for (i = 0; i < threadDim; i++) {
        if (check_val[i] != 0) {
            check_result = false;
            printf("thread %d check_matmul failed, returns %d\n", i, *check_val);
        }
    }


    if (check_result) {
        printf("all check_matmul checked done\n");


        // 在Host模拟矩阵乘操作，并将结果存放到h_output_reference
        half *h_output_reference = (half *)malloc(M * N * sizeof(half));
        memset(h_output_reference, 0, M * N * sizeof(half));
        for (i = 0; i < M; i++) {
            for (j = 0; j < N; j++) {
                for (k = 0; k < K; k++) {
                    h_output_reference[i * N + j] += h_input[i * K + k] * h_weight[k * N + j];
                }
            }
        }


        // 将Device端矩阵乘计算结果与Host端计算结果进行对比，验证失败则打印对应日志
        for (i = 0; i < M * N; i++) {
            if ((h_output[i] - h_output_reference[i] > 1.5e-2)) {
                printf("find mismatching data Pos %d, on line %d, col %d: ace: %f, serial: %f\n",
                          i, i / N, i % N, (float)h_output[i], (float)h_output_reference[i]);
                break;
            }
            if (i == M * N - 1) {
                printf("SUMMA result compare ok\n");
            }
        }


        // 释放h_output_reference空间
        free(h_output_reference);
    }


    // 释放在Device端和Host申请的内存
    free(check_val);
    sdaaFree(dev_outputheck_val);
    sdaaFree(dev_output);
    sdaaFree(dev_weight);
    sdaaFree(dev_input);
    free(h_output);
    free(h_weight);
    free(h_input);


    return 0;
}
```

## 15.4 使用CAS完成自定义原子操作

### 15.4.1 概述

CAS（Compare And Swap，比较并交换）是一种无锁机制，也称为乐观锁。比较Global存储空间上指定内存位置的值与预期值是否相等，如果相等，则将新值赋予指定内存，如果不相等，则保持目标内存的值不变。当多个SPE同时对同一个内存地址的值进行操作时，CAS可以保证只有一个SPE能够成功更新数据，因此，通过CAS可以实现原子操作。

本例使用CAS完成浮点类型的原子加操作。

程序分为主机端和设备端，主机端负责设备调用，空间申请及释放，数据传输；设备端主要负责计算。

### 15.4.2 编程思路

#### 主机端

1. 分配主机端内存 `host_a` 并初始化。
2. 主机端与设备端的数据传输。
   a. 设置计算核心阵列SPA号。
   b. 分配Global存储空间 `dev_a`。
   c. 从主机端内存 `host_a` 向Global存储空间 `dev_a` 传输数据。
   d. 调用Kernel函数进行计算。
   e. 将计算结果从Global存储空间 `dev_a` 传回主机端内存 `host_a`。
3. 打印结果并释放Global存储空间 `dev_a`。

#### 设备端

1. 利用函数接口 `atomic_cas_bool`，实现浮点类型的原子加函数。
   a. 创建变量 `address`，用于转换指针类型，以便传入 `atomic_cas_bool`。
   b. 创建变量 `old`，用于获取目标内存的数值。
   c. 创建变量 `is_ok`，用于判断交换是否成功。
   d. 构建循环，创建浮点变量 `expect`，保存期望的浮点加结果。调用 `atomic_cas_bool` 进行交换：
      - 满足交换条件：交换 `expect` 到目标内存，`atomic_cas_bool` 接口返回 true。
      - 不满足交换条件时：`atomic_cas_bool` 接口返回 false。继续循环，直到完成一次交换。
2. 利用步骤1的原子加函数，对 `dev_a` 的数据做运算。

### 15.4.3 编写程序

#### 15.4.3.1 主机端

##### 主机端程序设计思路

1. 分配主机端内存 `host_a` 并初始化。

   为了简单展示程序效果，这里使用一个浮点型数组存储两个数字，一个用于原子计算，一个普通计算，两个结果用于对比。

   ```c
   #define SIZE 2
   float host_a[SIZE] = {0, 0};
   ```

2. 主机端与设备端的数据传输。

   a. 设置计算核心阵列SPA号：通过 `sdaaSetDevice` 接口指定Kernel函数运行在哪一个计算核心阵列上。

   ```c
   sdaaSetDevice(0);
   ```

   b. 分配Global存储空间：通过 `sdaaMalloc` 接口在设备端分配给 `dev_a`。

   ```c
   int size = SIZE * sizeof(float);
   float *dev_a = NULL;
   sdaaMalloc((void**)(&dev_a), size);
   ```

   c. 数据传输：通过 `sdaaMemcpy` 接口将Host端内存 `host_a` 对应拷贝到Device端Global存储空间 `dev_a`。

   ```c
   sdaaMemcpy(dev_a, &host_a, size, sdaaMemcpyHostToDevice);
   ```

   d. 调用Kernel函数：通过 `<<<…>>>` 调用Kernel函数。

   ```c
   calc<<<1, stream>>>(dev_a);
   ```

   e. 拷贝计算结果：通过 `sdaaMemcpy` 接口将Device端Global存储空间 `dev_a` 中的数据拷贝到Host内存 `host_a`。

   ```c
   sdaaMemcpy(&host_a, dev_a, size, sdaaMemcpyDeviceToHost);
   ```

3. 打印结果并释放Global存储空间 `dev_a`。

   a. 使用 `printf` 打印 `host_a` 中的数据。

   ```c
   printf("host_a[0]=%f, host_a[1]=%f\n", host_a[0], host_a[1]);
   ```

   b. 释放申请的Device端Global存储空间：通过 `sdaaFree` 接口释放Device端的Global存储空间。

   ```c
   sdaaFree(dev_a);
   ```

##### 主机端程序

```c
#include <stdio.h>
#include <stdlib.h>
#define SIZE 2
int main()
{
    float host_a[SIZE] = {0, 0};


    // 指定Kernel函数运行在0号计算核心阵列SPA上
    sdaaSetDevice(0);
    // 在设备端的Global存储空间分配大小为size的连续内存
    int size = SIZE * sizeof(float);
    float *dev_a = NULL;
    sdaaMalloc((void**)(&dev_a), size);


    // 将主机端内存host_a的数据拷贝到设备端Global存储空间，大小为size
    sdaaMemcpy(dev_a, &host_a, size, sdaaMemcpyHostToDevice);


    // 调用名为calc的核函数，在设备端执行计算任务。
    sdaaStream_t stream;
    sdaaStreamCreate(&stream);
    calc<<<1, stream>>>(dev_a);
    sdaaDeviceSynchronize();


    // 将Global存储空间dev_a的数据拷贝到主机端host_a，大小为size
    sdaaMemcpy(&host_a, dev_a, size, sdaaMemcpyDeviceToHost);
    printf("host_a[0]=%f, host_a[1]=%f\n", host_a[0], host_a[1]);


    // 释放dev_a指针指向的Global存储空间
    sdaaFree(dev_a);


    return 0;
}
```
