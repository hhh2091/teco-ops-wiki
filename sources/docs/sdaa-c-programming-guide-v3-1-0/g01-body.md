# SDAA C 编程指南 v3.1.0（第g01部分）

## 1.2 什么是 SDAA C？

随着计算机技术的飞速发展，传统的 CPU 处理器体系已经无法满足天气预报、工程仿真和以深度学习为代表的人工智能等领域对计算资源的需求，从而促使计算机的处理器体系发生从单核到多核再到众核的革命性跨越。

目前，以众核作为加速核心的主流产品包括：NVIDIA GPU 和 AMD GPU 等，对应的编程语言分别为：CUDA C/CUDA
